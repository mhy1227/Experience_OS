# Experience OS 版本规划

> 来源:001_PRD/version、001_PRD/lark(ChatGPT 建议)+ 已落地原型
> 写法:**能力演进 ×平行技术线**(产品路线图,非功能堆砌)

## 产品定位

**Experience OS —— 个人经验管理与决策辅助引擎(Personal Experience Engine)。**

- **采集层(Capture)**:飞书 / 微信 / 浏览器插件 / App …—— 负责"记录",**是入口,不是产品**。
- **智能层(Intelligence)= Experience OS**:负责**提炼 → 关联 → 验证 → 决策辅助**。这层才是护城河。

> 一句话:**让每一次经历,都变成下一次决策的资产。**
> 我们解决的不是"今天发生了什么",而是"如何避免明天重复犯同样的错"。

## 两条演进线

| 产品能力演进 | 技术能力演进 |
|------|------|
| V1 经验提炼 | 本地存储 |
| V2 经验关联(规律发现) | 用户自定义 API Key / 多模型 |
| V3 经验验证 | 多模型支持 |
| V4 决策辅助 | 本地向量检索 |
| V5 组织经验 | 飞书生态接入 |

---

## 版本详情

### V1 经验提炼(比赛版本)
- **目标**:把碎片化、口语化经历转成结构化经验。
- **能力**:自由输入 / AI 经验提炼 / 自动标签分类 / 经验库沉淀 / 周·月复盘。
- **原则**:Local First(本地优先)· AI Assisted(AI 辅助提炼,不替代思考)· User Ownership(经验归用户)。

### V2 经验关联(规律发现)
- **目标**:发现经验之间的联系。
- **能力**:相似经验聚类 / 高频问题识别 / 成功模式识别 / 时间维度分析。
- **价值**:让用户看见自己难以察觉的重复模式("过去 90 天:目标不清晰 12 次")。

### V3 经验验证
- **目标**:让经验从"观点"变成"经过验证的策略"。
- **能力**:验证次数统计 / 成功率分析 / 可信度评分 / 策略演化记录。
- **价值**:建立属于自己的高可信决策库。

### V4 决策辅助
- **目标**:在未来关键场景主动调用历史经验。
- **能力**:场景识别 / 风险提醒 / 历史经验召回 / 决策建议。
- **价值**:减少重复犯错("这门高价课你过去买过 6 次,完成率仅 25%")。

### V5 组织经验(生态)
- **目标**:从个人经验网络扩展到团队/组织经验网络。
- **能力**:飞书机器人接入 / 多维表格同步 / 团队经验库 / 组织知识资产。
- **价值**:个人经验向组织经验传递与复用。

## 当前实现范围(本阶段只做"部分")

> 原型已跨越 V1–V4 的基础能力,本阶段聚焦把 **V1 + V2** 打磨成可演示的比赛核心;V3 引擎已存在(暂折叠),V4 已有基础版,**V5/飞书明确推迟**。

| 能力 | 当前状态 | 说明 |
|------|:--:|------|
| V1 经验提炼 | ✅ 已实现 | 真模型(DeepSeek)提炼 + 三态分类(策略/避坑/待观察)+ 经验库 |
| V1 周·月复盘 | ✅ 已实现 | `periodicReview` 本周/本月摘要(记录数 + 高频问题/成功 + 一句话建议),规律发现面板内 |
| V2 规律发现 | ✅ 基础版 | `patternDiscovery` 统计聚类 + 模型归因 + 「扫描 90 天」归因卡 |
| V3 经验验证 | 🟡 引擎已存在,暂隐藏 | `evaluationEngine`(复测/可信度/成功率)折叠为"高级面板",按需作可信度后端 |
| V4 决策辅助 | 🟡 基础版 | `decisionHints` 录入时召回历史规则提醒(阈值/可忽略) |
| V5 组织/飞书 | ⬜ 推迟 | 采集层未来项,比赛不做 |
| 技术:本地存储 | ✅ | localStorage 为唯一真相源 |
| 技术:自定义 Key/多模型 | ✅ 基础版 | DeepSeek 内置 + 用户可配;多家厂商接口已抽象 |
| 技术:向量检索 / 飞书 | ⬜ 推迟 | V4/V5 再做 |

### V1 非目标(本阶段坚决不做)
团队知识库 · 多人协作 · AI 教练 · 第二大脑 · RAG 平台 · MCP 生态 · 飞书接入 —— 全部推迟到 V5+。

### V1 缺口已补齐
**周·月复盘摘要**(`src/services/periodicReview.ts`):按周/月窗口统计"记录 N 条 / 高频问题 / 高频成功 / 一句话建议",已接入规律发现面板。至此 **V1 能力闭环**(采集→提炼→经验库→复盘→规律发现)。
